# Cadastro-Em-Django
Projeto Acadêmico de introdução ao Django

# 👥 Projeto de Cadastro de Usuários

Este é o meu primeiro projeto desenvolvido em **Django**. O objetivo principal do sistema é gerenciar o fluxo básico de um cadastro de usuários, aplicando conceitos fundamentais do framework como rotas (URLs), Views, Models e renderização de templates HTML integrados com o ecossistema do Django.

---

## 🚀 Tecnologias Utilizadas

*   **Python 3.14+**
*   **Django 6.0+** (Framework Web Back-end)
*   **Bootstrap 5.0** (Estilização e Responsividade Front-end)
*   **SQLite** (Banco de dados nativo do desenvolvimento Django)
